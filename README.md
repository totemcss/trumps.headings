# Headings

inuitcss plugin based on @inuitcss' [trumps-headings](https://github.com/inuitcss/trumps.headings)

## Installation

- npm : `npm install --save totem-trumps-headings`
- bower : `bower install --save totem-trumps-headings`

## Available classes

- `u-alpha`
- `u-beta`
- `u-gamma`
- `u-delta`
- `u-epsilon` 
- `u-zeta`
